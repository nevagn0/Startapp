# Тема "Сайт для поиска тиммейтов"

## 1. Финансы

### Затраты:
- Сервера
- Заработная плата (ежемесячно)

### Доход:
- Подписка (редактирование профиля)
- Реклама

## 2. Проблема

В текущей ситуации проблематично найти хорошего тиммейта для своей группы.

## 3. Решение проблемы

Создание сайта для поиска тиммейтов с использованием SocialRating для избежания токсичных тиммейтов.

## 4. Анализ

| Сервис       | Сильные стороны              | Слабые стороны                  | Отличие нашего проекта |
|--------------|------------------------------|---------------------------------|----------------------|
| **Discord**  | Удобный чат, голосовые каналы | Заблокирован в РФ, нет поиска тиммейтов | Специализация только на поиске |
| **TeamSpeak**| Низкие требования, стабильность | Устаревший интерфейс, нет поиска | Современный веб-интерфейс |
| **Game3**    | Популярность в СНГ           | Нет поиска, нет системы SocialRating | Встроенный рейтинг игроков |

# Тема: "Карта тишины"

## 1. Финансы

### Доход:
- Агенты недвижимости
- Подписка (уведомление по адресу, экспорт в PDF)

### Затраты:
- Датчики шума / обслуживание
- Сервера

## 2. Проблема

Нет сервиса, который показывает «где тихо» в городе в реальности.

## 3. Решение проблемы

Интерактивная карта города:
- Зелёные зоны — тихо
- Жёлтые — умеренно
- Красные — шумно
- Коричневые — «опасно для сна»

## 4. Анализ

| Сервис | Сильные стороны | Слабые стороны | Отличие нашего проекта |
|--------|----------------|----------------|------------------------|
| **Яндекс.Карты** | Транспортная доступность, пробки в реальном времени, миллионы пользователей | Нет слоя «шум», нет краудсорсинга тишины, нет фильтрации по времени суток | Узкая специализация — мы делаем только карту тишины, а не всё подряд |
| **EcoStandard group** | Профессиональные замеры, сертифицированное оборудование, доверие юрлиц | Стоимость от 50 000 ₽, только B2B, нет публичной карты | Народная карта — бесплатно, доступно всем, краудсорсинг |
| **Жалобы в соцсетях** | Живая реакция, реальные люди, оперативность | Тонут в ленте, нет геопривязки, не структурированы | Систематизация — превращаем хаос жалоб в упорядоченную карту |
| **Карты шума (официальные)** | Точные данные, научный подход | Обновляются раз в 5 лет, только крупные города, не учитывают бытовой шум | Актуальность 24/7 — данные сегодня, а не в 2019 году |
