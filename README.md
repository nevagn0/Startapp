## Тема "Сайт для поиска тиммейтов"

### 1. Финансы

#### Затраты:
- Сервера
- Заработная плата (ежемесячно)

#### Доход:
- Подписка (редактирование профиля)
- Реклама

### 2. Проблема

В текущей ситуации проблематично найти хорошего тиммейта для своей группы.

### 3. Решение проблемы

Создание сайта для поиска тиммейтов с использованием SocialRating для избежания токсичных тиммейтов.

### 4. Анализ

| Сервис       | Сильные стороны              | Слабые стороны                  | Отличие нашего проекта |
|--------------|------------------------------|---------------------------------|----------------------|
| **Discord**  | Удобный чат, голосовые каналы | Заблокирован в РФ, нет поиска тиммейтов | Специализация только на поиске |
| **TeamSpeak**| Низкие требования, стабильность | Устаревший интерфейс, нет поиска | Современный веб-интерфейс |
| **Game3**    | Популярность в СНГ           | Нет поиска, нет системы SocialRating | Встроенный рейтинг игроков |

## Тема: "Карта тишины"

### 1. Финансы

#### Доход:
- Агенты недвижимости
- Подписка (уведомление по адресу, экспорт в PDF)

#### Затраты:
- Датчики шума / обслуживание
- Сервера

### 2. Проблема

Нет сервиса, который показывает «где тихо» в городе в реальности.

### 3. Решение проблемы

Интерактивная карта города:
- Зелёные зоны — тихо
- Жёлтые — умеренно
- Красные — шумно
- Коричневые — «опасно для сна»

### 4. Анализ

| Сервис | Сильные стороны | Слабые стороны | Отличие нашего проекта |
|--------|----------------|----------------|------------------------|
| **Яндекс.Карты** | Транспортная доступность, пробки в реальном времени, миллионы пользователей | Нет слоя «шум», нет краудсорсинга тишины, нет фильтрации по времени суток | Узкая специализация — мы делаем только карту тишины, а не всё подряд |
| **EcoStandard group** | Профессиональные замеры, сертифицированное оборудование, доверие юрлиц | Стоимость от 50 000 ₽, только B2B, нет публичной карты | Народная карта — бесплатно, доступно всем, краудсорсинг |
| **Жалобы в соцсетях** | Живая реакция, реальные люди, оперативность | Тонут в ленте, нет геопривязки, не структурированы | Систематизация — превращаем хаос жалоб в упорядоченную карту |
| **Карты шума (официальные)** | Точные данные, научный подход | Обновляются раз в 5 лет, только крупные города, не учитывают бытовой шум | Актуальность 24/7 — данные сегодня, а не в 2019 году |

## Помощь в получении гражданства РФ

### Описание  
Приложение-помощник, которое пошагово сопровождает пользователя на пути к получению гражданства РФ, заменяя дорогие юридические консультации и помогая ориентироваться в постоянно меняющемся законодательстве.

### Зачем  
Процедура получения гражданства сложная и запутанная:
- разные основания и сценарии  
- большое количество документов  
- частые изменения законов  
- высокая стоимость услуг юристов  

Пользователю нужен понятный и актуальный инструмент.

### Как работает  
1. Пользователь отвечает на вопросы:
   - текущее гражданство  
   - основание для получения  
   - текущий статус (нет документов / РВП / ВНЖ)  
2. Приложение определяет оптимальный путь  
3. Формируется персональный пошаговый план  
4. Пользователь получает:
   - чек-листы документов  
   - инструкции куда и когда идти  
   - напоминания о сроках  
5. При изменении законодательства логика автоматически обновляется

### Ключевые возможности  
- Пошаговая навигация до паспорта РФ  
- Актуальные требования и сроки  
- Уведомления и напоминания  
- Простое объяснение юридических терминов  
- Предупреждение об ошибках  

### Монетизация  
- Бесплатно: основной функционал и обновления  
- Платно:
  - проверка документов  
  - консультации  
  - персональные рекомендации

## Оптимизатор походов по продуктовым магазинам

### Описание  
Приложение, которое строит оптимальный маршрут по магазинам для покупки всех нужных продуктов с минимальными затратами времени и усилий.

### Зачем  
Проблемы обычного похода за продуктами:
- лишние километры  
- тяжёлые пакеты  
- хаотичные маршруты  
- потраченное время  

### Как работает  
1. Пользователь формирует корзину товаров  
2. Указывает область на карте (район / радиус)  
3. Приложение находит магазины с нужными товарами  
4. Строит оптимальный маршрут по выбранному сценарию  

### Варианты оптимизации  
- Самый быстрый маршрут  
- Сначала дальние магазины  
- Минимум переносимого веса  
- Минимальная стоимость  

### Ключевые возможности  
- Интерактивная карта  
- Несколько вариантов маршрутов  
- Замена отсутствующих товаров  
- История покупок  

### Монетизация  
- Бесплатно: базовая маршрутизация  
- Платно:
  - учёт акций  
  - расширенные алгоритмы  
  - семейные корзины

## Калькулятор стоимости техники с прогнозом цены

### Описание  
Сервис для оценки оптимальной рыночной стоимости б/у техники и прогноза изменения цены по месяцам.

### Зачем  
Продавцы часто:
- завышают цену — товар не продаётся  
- занижают — теряют деньги  

### Как работает  
1. Пользователь вводит:
   - тип устройства  
   - характеристики  
   - год выпуска  
   - состояние и дефекты  
2. Приложение анализирует рынок  
3. Выдаёт:
   - оптимальную цену  
   - диапазон торга  
   - прогноз стоимости  

### Ключевые возможности  
- Учёт износа и дефектов  
- Анализ спроса  
- График изменения цены  
- Рекомендация по срокам продажи  

### Монетизация  
- Бесплатно: базовая оценка  
- Платно:
  - расширенный прогноз  
  - рекомендации площадок  
  - автопубликация объявлений

## Онлайн-карта кладбища

### Описание  
Цифровая система учёта захоронений и поиска могил на карте для похоронного бизнеса и обычных людей.

### Зачем  
Многие кладбища до сих пор:
- ведут учёт в тетрадях  
- теряют данные  
- не могут быстро найти захоронение  

### Как работает  
- Интерактивная карта кладбища  
- Каждое захоронение — точка с данными  
- Поиск по ФИО, датам, участкам  
- Онлайн-бронирование мест  

### Ключевые возможности  
- Поиск могил родственников  
- Фото и история захоронений  
- Административный интерфейс  
- Цифровой архив  

### Монетизация  
- SaaS для кладбищ и агентств  
- Платные семейные страницы  
- Расширенное хранение данных

## Оптимальный поиск товара в городе

### Описание  
Приложение для поиска конкретных товаров в офлайн-магазинах города с отображением наличия и расстояния.

### Зачем  
Поиск редких товаров офлайн занимает много времени:
- обзвон магазинов  
- нет данных о наличии  

### Как работает  
1. Пользователь вводит название товара  
2. Приложение ищет магазины в городе  
3. Показывает:
   - список магазинов  
   - расстояние  
   - цену  
   - наличие (при наличии API)  

### Ключевые возможности  
- Поиск по карте  
- Фильтры по расстоянию и цене  
- Уведомления о появлении товара  

### Монетизация  
- Бесплатно: базовый поиск  
- Платно:
  - уведомления  
  - история цен  
  - партнёрские интеграции

## Путешествие через одно приложение

### Описание  
Единая платформа для полного цикла путешествия — от билетов до экскурсий.

### Зачем  
Путешественникам приходится пользоваться десятками сервисов, теряя информацию и контроль.

### Как работает  
Пользователь:
1. Планирует маршрут  
2. Покупает билеты  
3. Бронирует жильё  
4. Заказывает экскурсии и трансферы  

Вся информация хранится в одном месте.

### Ключевые возможности  
- Авиабилеты и отели  
- Экскурсии и активности  
- Уведомления и напоминания  
- Хранение бронирований  

### Монетизация  
- Комиссии с бронирований  
- Подписка на премиум-функции  
- Партнёрские программы
